# SKILL Template

Keep SKILL.md focused. Use `doc/` for long examples/templates.

Minimum sections (recommended):

- YAML front matter: `name`, `description`
- Aim: 1–3 lines
- Core: 3–5 bullets
- Do & Examples: concise ✅/❌ pairs
- Edge: when to deviate
- Refs: URL + `(2026-01-26)`

Notes:
- Write content in English.
- Prefer short, actionable rules; move lengthy rationale into `doc/`.
- Use relative paths for internal refs (e.g. `doc/template.md`).

Example minimal front matter:

```yaml
---
name: example_skill
description: Short one-line description when to use
---
```

For a detailed long-form template, see `skills_creator/SKILL.md` or `skills/skills_creator/doc/patterns.md`.
