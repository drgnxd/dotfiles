# Research Tools

## search_and_analyze.py
Usage: `uv run python search_and_analyze.py --query "topic" --output findings.md`

## synthesize.py
Usage: `uv run python synthesize.py --input file1.md,file2.md --output synthesis.md`

## report_generator.py
Usage: `uv run python report_generator.py --synthesis synthesis.md --output report.md`

Refer to ../specialized/research.yaml for workflow details.
