# Coding Tools

## check_style.sh
Usage: `./check_style.sh path/to/code`

Optional: `--allow-missing` to skip unavailable tools.

## lint.py
Usage: `uv run python lint.py path/to/code`

Optional: `uv run python lint.py --allow-missing path/to/code`

Refer to essential/practices.yaml and skills_core.yaml for conventions.
