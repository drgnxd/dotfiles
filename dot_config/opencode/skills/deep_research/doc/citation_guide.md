# Citation Guide

Use consistent references to maintain traceability.

## Reference Entry Format

```
[Author] ([Year]). [Title]. [Publisher]. [URL]. Accessed: YYYY-MM-DD.
```

If the author is unknown, use the organization name. If the year is unknown, use "n.d.".

## In-Text References

- Use short references in parentheses: (Author, Year)
- For numbered references, use [1], [2], etc.

## Minimum Fields to Record

- Author or organization
- Publication date
- Title
- Publisher or outlet
- URL
- Access date

## Example

```
OpenAI (2024). GPT-4 Technical Report. OpenAI. https://openai.com/research/gpt-4. Accessed: 2026-01-26.
```
