# Common Conventions

## Naming
All skills follow `default_naming_conventions/doc/naming_protocol.md` unless language/framework mandates different conventions.

## Reference Format
URLs: `https://example.com (YYYY-MM-DD)`

## Edge Cases
- Project rules > global defaults
- Document exceptions w/ rationale
- Prefer explicit over implicit

## Standard Sections
Skills may include:
- Purpose, Core Principles, Rules, Examples, Edge Cases, References
- Link detailed content in `doc/` subdirectories
